# Сайт Полины Шапкиной

Минималистичный сайт-портфолио художницы на React + Vite.

## Как запустить локально

1. Установить Node.js: https://nodejs.org/
2. Открыть папку проекта в терминале.
3. Выполнить команды:

```bash
npm install
npm run dev
```

После этого сайт откроется по локальной ссылке, которую покажет терминал.

## Как бесплатно опубликовать на Vercel

1. Создать аккаунт на GitHub: https://github.com/
2. Создать новый репозиторий.
3. Загрузить в него все файлы из этой папки.
4. Создать аккаунт на Vercel: https://vercel.com/
5. Нажать **Add New Project**.
6. Выбрать GitHub-репозиторий с сайтом.
7. Нажать **Deploy**.

Vercel автоматически соберёт сайт и даст публичную ссылку.

## Где менять контент

Основной контент находится в файле:

```txt
src/App.jsx
```

Изображения лежат здесь:

```txt
public/images/
```

Чтобы заменить изображение, добавьте файл в `public/images/` и укажите путь в `src/App.jsx`, например:

```js
image: "/images/new-work.jpg"
```
